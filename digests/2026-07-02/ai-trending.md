## 趋势信号分析

**Agent 生态已从 "框架之争" 进入 "基础设施分层" 阶段。** 今日 Trending 中能清晰看到 Agent 栈的纵切分层正在形成：上层是 `agency-agents`（角色模板库）、`Vibe-Trading`/`video-use`（垂直 Agent 产品）；中层是 `herdr`（终端多路复用）、`OmniRoute`（多模型路由）、`headroom`/`claude-mem`/`mem0`（上下文与记忆）；底层是 `TencentCloud/CubeSandbox`（沙箱）、`logto`（鉴权）。这一分层结构与 2024 年的 LLM 应用栈高度相似，预示 Agent Runtime 正走向模块化、标准化。

**新兴方向 "AI + 安全" 首次大规模登榜。** `strix` 单日 +1211 stars，`VulnClaw`、`council-of-high-intelligence`（多模型对抗决策）共同指向 "用 AI 做渗透 / 用多 AI 做博弈决策" 这一蓝海。其底层驱动力是 Claude/GPT-5 系列在工具调用与代码生成上的飞跃，让自动化攻防成为可能；与传统密码工具 `cupp` 形成鲜明对比，AI 化安全工具正在改写安全社区格局。

**"Vibe Coding" 范式向更多垂直领域扩散。** 从 `Vibe-Trading`（交易）到 `video-use`（视频编辑），"自然语言意图 → Agent 执行" 的产品形态已脱离纯编程场景，进入金融、内容创作、设计（`facebook/astryx`）。这与近期大模型在长链路工具调用上的显著进步直接相关，也意味着新一轮 AI 应用层的 "场景争夺战" 即将展开。
